# Block OS Suite v1.1
Custom tools and utilities for **Block OS**, a portable Linux distribution based on LFS 13.0.

## Included Tools:
*   **be**: Native Go text editor with alternate buffer and status bar.
*   **bhub**: Python system dashboard (auto-launches at login).
*   **bstats**: Real-time CPU/RAM monitor with ASCII block visuals.
*   **bls**: Themed file explorer with "Block" sized outputs.
*   **bcode**: Host-side developer tool for building and deploying to Block OS.
*   **bpm**: The Block Package Manager (v0.3 with network fetch support).

## Deployment
Use `bcode` on your host to build and deploy any of these tools to your Block OS VM via SSH.
